## 실행 방법
<pre><code> npm i </code></pre>
<pre><code> npm run dev </code></pre>

## 디렉토리 구조
- 도메인 단위로 폴더를 나눠서 작업
- domain -> 도메인 별 폴더 존재
- global -> 전역 설정, 테스트 등 전역으로 영향을 미치는 파일
